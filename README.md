# DreamCat

**安装**
------

>尝试版本(可能有Bug)  
1.请点击Code下载  
2.放入Typecho "themes"文件夹中  
3.修改本主题文件夹名为"DreamCat"  
4.启用以完成安装  

>稳定版本  
1.请点击Releases下载稳定版本  
2.放入Typecho主题文件夹中  
3.启用以完成安装
