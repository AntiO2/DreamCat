## 安装(Install)

> ### 开发版本（最新）
> 1. 建议使用`Git Clone`以动态获取后续版本 (或点击 Code 下载)
> 2. 放入`Typecho` `usr/`目录下`themes`文件夹中
> 3. 请确保本主题文件夹名为`DreamCat`
> 4. 在`Typecho 管理页面`->`网站外观`启用以完成安装

> ### 稳定版本
> 1. 请点击`Releases`下载稳定版本
> 2. 完全解压并放入`Typecho` `usr/`目录下`themes`文件夹中
> 3. 请确保本主题文件夹名为`DreamCat`
> 4. 在`Typecho 管理页面`->`网站外观`启用以完成安装
